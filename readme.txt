== Description ==

Renalinfolk is a professional WordPress block theme designed specifically for pediatric nephrology departments, medical practices, and healthcare institutions. Built on WordPress's Full Site Editing (FSE) capabilities, it emphasizes trust, medical expertise, and family-friendly design through carefully selected colors, soft rounded geometries, and clear information hierarchy.

**Key Features:**

* **Medical Design System:** Professional blue and green color palette optimized for trust and credibility in medical contexts
* **Family-Friendly Aesthetics:** Soft rounded borders and welcoming visual elements appropriate for pediatric care
* **Accessibility Ready:** WCAG AA compliant with high contrast ratios and clear typography hierarchy
* **Full Site Editing:** Complete control over your site's design using WordPress's block editor
* **Medical-Optimized Patterns:** Pre-built layouts for articles, publications, testimonials, patient resources, and medical services
* **Lexend Typography:** Professional, highly readable font family perfect for medical content
* **Dark Mode Variation:** Alternative style variation for user preference
* **Responsive Design:** Optimized for all devices from mobile to desktop
* **Block Pattern Library:** 98+ customizable patterns including medical pages, content sections, and service layouts

**Perfect For:**

* Pediatric nephrology departments
* Hospital and clinic websites
* Medical research publications
* Patient education resources
* Healthcare professional portfolios
* Medical staff directories
* Department announcements and news

**Color Palette:**

* Primary Blue (#135bec) - Interactive elements and links
* Primary Dark Green (#004d40) - Headers and emphasis
* CTA Yellow (#FFC300) - Call-to-action buttons
* Green-Blue (#006D77) - Accent elements
* Footer Dark (#1C2541) - Footer backgrounds
* Primary Dark Blue (#1a237e) - Gradient accents

**Typography:**

* Primary Font: Lexend (Variable weight 400-900)
* Monospace Font: Fira Code (Variable weight 300-700)
* Fluid typography with responsive scaling
* Bold headings (weight 800) for clear hierarchy
* Comfortable line height (1.5) for medical content

== Installation ==

1. In your WordPress admin dashboard, go to Appearance > Themes > Add New
2. Click Upload Theme and choose the renalinfolk.zip file
3. Click Install Now
4. Once installed, click Activate to make Renalinfolk your active theme

Alternatively, upload the theme folder to /wp-content/themes/ directory via FTP/SFTP.

== Customization ==

**Global Styles:**
Navigate to Appearance > Editor > Styles to customize colors, typography, and spacing globally. Choose between Light Mode (default) and Dark Mode style variations.

**Block Patterns:**
Access ready-made medical content patterns via the block inserter (+) in the editor. Categories include:
* Medical Pages - Full page layouts
* Medical Content - Content sections
* Testimonials - Patient stories
* Services - Treatment options
* Calls-to-Action - Newsletter signups, resource downloads

**Header & Footer:**
Edit template parts via Appearance > Editor > Patterns > Template Parts. The theme includes:
* Gradient header with navigation
* Dark footer variations
* Multi-column footer layouts

**Recommended Plugins:**

* Contact Form 7 or WPForms - For patient inquiry forms
* Yoast SEO - For medical content optimization
* WP Accessibility - For enhanced accessibility features
